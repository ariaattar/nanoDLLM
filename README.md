## nanoLLM is a minimal PyTorch implementation of a Diffusion Language Model, adapted from the "Dream" paper:  
https://arxiv.org/html/2508.15487v1

---

## Dataset

Get the [Tiny Shakespeare corpus](https://raw.githubusercontent.com/karpathy/char-rnn/master/data/tinyshakespeare/input.txt):

```bash
wget https://raw.githubusercontent.com/karpathy/char-rnn/master/data/tinyshakespeare/input.txt
```
